Embed Code

Use this code to embed the compact roulette with maximum size and no title:

<iframe src="embed.html?size=500&showControls=compact&title=" width="800" height="600" frameborder="0" allowtransparency="true"> </iframe>

Parameters Used:

  • size=500 - Maximum size (500px) for the roulette wheel
  • showControls=compact - Compact controls with hamburger menu
  • title= - Empty title (no title displayed)
