URL Parameters

You can customize the roulette using URL parameters:

Parameter Description Example Default
size Size of the roulette wheel in pixels ?size=300 300
snacks Comma-separated list of options ?snacks=Pizza,Burger,Pasta Dutch snacks
showControls Control display mode: hidden, compact, or full ?showControls=compact compact
title Title text (empty string for no title) ?title=My Roulette üçø Snack Roulette üçø
