How to Embed

To embed the compact roulette in your website, you can use it in several ways:

1. Basic Usage

<CompactRoulette size={300} />

2. With Custom Snacks

<CompactRoulette size={250} snacks={['Pizza', 'Burger', 'Pasta', 'Sushi', 'Tacos']} />

3. With Controls

<CompactRoulette size={350} showControls={true} snacks={['Option 1', 'Option 2', 'Option 3']} />
Note: The compact roulette is designed to be self-contained and responsive. It automatically adjusts text size and layout based on the size prop. The confetti animation is included and will trigger when a result is selected.
